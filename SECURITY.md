# Security Policy

This repository is a defensive, offline-only educational lab for synthetic alert
data. It must not be used against real systems, real customer data, real
credentials, or live networks.

## Safety Boundaries

- Synthetic/local alert data only.
- No network calls.
- No live scanning.
- No exploit code.
- No external AI API.
- No external threat intelligence API.
- No offensive tooling integrations.

## Reporting Issues

Report issues that weaken the offline-only or synthetic-data safety model, such
as unsafe examples, accidental network-capable behavior, or documentation that
suggests using real customer data.

Do not include real secrets, tokens, credentials, private alerts, or customer
data in issue reports.
