# Portfolio Notes

This project demonstrates defensive security engineering with strict safety
boundaries.

Key points to highlight:

- Offline-only local processing.
- Synthetic alert data only.
- Deterministic rule-based "AI-assisted" triage.
- No external AI API, LLM, API keys, live scanning, exploit code, or threat
  intelligence calls.
- Transparent, traceable decisions.
- Markdown and JSON reporting.
- pytest coverage, Ruff checks, docs safety checks, CLI smoke, and locally
  configured GitHub Actions, CodeQL, and Dependabot.
- Hosted GitHub CI and hosted CodeQL still need verification after publishing.

The portfolio framing should emphasize safe lab design and deterministic
analysis rather than offensive capability.
